# FUTURE_ML_01
MACHINE LEARNING TASKS AND ASSIGNMENTS

SALES FORECASTING USING MACHINE LEARNING:

This project focuses on predicting future sales using historical business data. The aim of the project is to understand how machine learning can help businesses estimate future demand based on past sales patterns.

PROJECT OBJECTIVE:

The main objective of this project is to build a model that can forecast future sales using previous sales data and some related factors. Forecasting sales helps businesses plan their inventory, manage stock, and make better decisions.

DATASET:

The dataset used in this project contains information such as store details, weekly sales, temperature, fuel price, CPI, and unemployment rate. These factors can influence customer purchasing behaviour and overall sales.

TOOLS AND TECHNOLOGIES USED:

For building this project, the following tools and libraries were used:

Python

Pandas for data handling

NumPy for numerical operations

Scikit-learn for building the machine learning model

Matplotlib for data visualization

Google Colab for running the project

MODEL USED:

In this project, a machine learning regression model was used to predict future sales based on historical data and economic indicators.

PROJECT OUTCOME:

The model predicts future sales for upcoming dates using the available data. The results are visualized using graphs so that trends and patterns can be easily understood.

BUSINESS USE:

Sales forecasting is very important for businesses. With the help of this model, a business owner or manager can:

estimate future product demand

plan inventory in advance

avoid overstocking or shortages

make better financial and operational decisions
